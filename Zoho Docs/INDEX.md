## ✨ Quick Starts by Task

### "I want to add a feature to the frontend"

1. QUICK_START.md → Run dev servers
2. frontend/DEVELOPMENT.md → Understand project structure
3. Create your component in `app/`
4. Use `npm run type-check` and `npm run lint:fix`

---

### "I want to add an API endpoint"

1. QUICK_START.md → Run dev servers
2. backend/DEVELOPMENT.md → Understand Express setup
3. Add endpoint to `src/index.ts`
4. Use `npm run type-check` and `npm run lint:fix`

---

### "I want to add a database model"

1. backend/DEVELOPMENT.md → Database section
2. Update `prisma/schema.prisma`
3. Run `npm run db:migrate`
4. Use in Express handlers

---

### "I want to add a dependency"

1. MONOREPO.md → Adding Dependencies
2. `npm install package --workspace=frontend`
3. Or `npm install package --workspace=backend`

---

### "I want to deploy"

1. SETUP_SUMMARY.md → Deployment section
2. Run `npm run build --workspaces`
3. Deploy frontend to Vercel
4. Deploy backend to hosting
